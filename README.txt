# Elegance Hall Event Center Website

Welcome to the **Elegance Hall Event Center** website repository! This project showcases a beautifully designed website for an event center, featuring information about upcoming events, catering services, and more.

## Features

- **Homepage**: A welcoming introduction to the event center.
- **Events Section**: Displays upcoming events with descriptions and images.
- **Menu Page**: Highlights catering options and movie night concessions.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Contact Page**: Allows users to inquire about bookings and services.

## Technologies Used

- **HTML**: For the structure of the website.
- **CSS**: For styling and layout.
- **Responsive Design**: Media queries for mobile-friendly layouts.

## How to View the Website

Responsive Design
The website is fully responsive and adapts to different screen sizes:

Desktop: Full layout with multiple columns.
Tablet: Adjusted layout for medium-sized screens.
Mobile: Single-column layout for smaller screens.
